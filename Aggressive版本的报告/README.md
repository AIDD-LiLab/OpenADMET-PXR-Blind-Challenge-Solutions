# Aggressive 版本的报告 (Aggressive Variant Report)

This directory will hold the report for the **aggressive submission variant** of the
OpenADMET PXR (NR1I2 / human PXR agonism, pEC50 regression) Blind Challenge, submitted
by **AIDD-LiLab** (University of Florida, Yanjun Li Lab).

The aggressive variant blends the **+440 high-activity-retrained components**
(SVM_20K + Holo + 3-way, kept at the original ensemble ratio) with the external
**qHTS discriminator**, as a deliberate Set-2 active-region bet.

📄 **The detailed report will be uploaded on July 1, 2026 (2026-07-01).**
